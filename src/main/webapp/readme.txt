должны быть установлены npm и node.js
делается в командной строке в директории /webapp.

собрать файл app-build.js :
npm run build

запустить фоновый процесс сборки app-build.js при редактировании :
npm run watch

сделать uglify app-build.js.
npm run uglify

uglify лучше делать перед выкатом на продакшен, так как постоянно использовать при отладке не удобно.

Названия команд и входящих/выходящих файлов настраиваеются в файле package.json в разделе "scripts"
